# Student Result Management System 📊

A Java-based console application to manage student results, calculate grades, and generate result sheets.

## 📌 About The Project
This system takes student names and marks as input, processes the data using arrays and conditional logic, and generates a formatted result sheet with grades and pass/fail status.

## 🎯 Key Features
- Add multiple students at once
- Auto calculates grade based on marks
- Displays pass/fail status
- Shows class average
- Tracks total passed and failed students
- Input validation for marks

## 🛠️ Technologies Used
- Java

## ▶️ How To Run
```bash
javac StudentResult.java
java StudentResult
```

## 📊 Grading System
| Marks | Grade |
|-------|-------|
| 90-100 | A+ |
| 80-89 | A |
| 70-79 | B |
| 60-69 | C |
| 50-59 | D |
| Below 50 | F (Fail) |
